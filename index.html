Kreiranje {Objekata}

  • Create an {object} from an [Array of Keys] and an [Array of Values] 
    var keys = ['foo', 'bar', 'baz'];
var values = [11, 22, 33]
var result = {};
keys.forEach((key, i) => result[key] = values[i]);
console.log(result);

Konverzija [niza] u {objekat}

  • Object.assign({}, ['a','b','c']); // {0:"a", 1:"b", 2:"c"}
  • { ...['a', 'b', 'c'] }
  • Object.fromEntries()

Konverzija {Objekata} u [Niz]

  • Object.keys(arr); // ['k1', 'k2']
  • Object.values(arr); // ['v1', 'v2']
  • Object.entries(arr); // [ ['k1', 'v1'], ['k2', 'v2'] ]

Duboko kopiranje jednostavnih objekata

  • JSON.stringify - JSON.parse 
    var objclone = JSON.parse(JSON.stringify(obj)) // najsporije sto se tice performansi
  • Manuelno 
    // (↑performanse↑, ↓komplexnost↓, ↓mora se znati struktira↓)
let obj = {
  id: 1,
  m: { male: true },
  i: { info: {name:"Jonh", lname:"Doe"}}
};

let objClone = {}
objClone = {...obj} // ovde je vec ceo objekat iskloniran, ALI NE DUBINSKI
objClone.m = {...obj.m}
objClone.i = {...obj.i}
objClone.i.info = {...obj.i.info}

console.dir('Befor Change - objClone: ', objClone);

obj.id = 2
obj.m.male = false
obj.i.info.name = "Joly";

console.dir('After Change - obj: ', obj);
console.dir('After Change - objClone: ', objClone);

Provere

  • da li {objekat} ne postoji 
    typeof objX === "undefined"
  • da li je {objekat} null 
    typeof objX !== "undefined" &&
        objX !== null
  • da li je Broj 
    !Number.isNaN(e) && typeof e === "number" && Number.isFinite(e) // true
  • Nulish: da li je null ili undefined, ako jeste vrati default vrednost 
    null ?? 5 // 5
undefined ?? 5 // 5
false ?? 5 // false
NaN ?? 5 // NaN
0 ?? 5 // 0
  • IN: da li postoji ključ: u {objektu} (ili index u [nizu] ). 
    "ime" in psi[0] // true
"god" in psi[0] // true
"god" in psi // false
"boja" in psi[0] // false
